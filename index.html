Ejercicios de JS

Primer ejercicio - Notas
  • Consigna

    Debemos crear un programa que pida una nota mediante un input,
    y muestre un mensaje con las siguientes condiciones:

    • ⚫ Mostrar "Aplazado", si la nota está entre 1 y 3 (inclusive)
    • ⚫ Mostrar "Desaprobado", si la nota está entre 4 y 6 (inclusive)
    • ⚫ Mostrar "Aprobado", si la nota está entre 7 y 10 (inclusive)

      Para esto crearemos:

    • ⚫ Una carpeta llamada nota
    • ⚫ Dentro de nota, crearemos un archivo index.html
  • Paso a paso
    • Paso 1

      • En index.html escribimos la estructura básica de la página web. Además, agregamos el tag script en el head


        Ver resultado
        <html>
            <head>
                 <script>

                 </script>
            </head>

            <body>

            </body>
        </html>
    • Paso 2

      • Dentro del body, crearemos un tag h1 para pedirle al usuario que ingrese una nota, y un tag input donde deberá ingresarla.
        Este ultimo debe tener un atributo id que sea nota.
        Tambien se creara un tag button con un atributo onclick que llame a la funcion mostrarResultado()


        Ver resultado
        <body>
            <h1> Por favor, ingrese una nota: </h1>

            <input id="nota" />
            <button onclick="mostrarResultado()"> Mostrar!: </button>
        </body>
    • Paso 3

      • Dentro del script, empezaremos a programar
        Arrancaremos con crear una funcion que se llame mostrarResultado


        Ver resultado
        <script>
            function mostrarResultado(){

            }
        </script>
    • Paso 4

      • Dentro de mostrarResultado vamos a guardar en una variable el contenido del input y crear una variable llamada resultado que por ahora no tendrá ningún valor definido.


        Ver resultado
        <script>
            function mostrarResultado(){
                var nota = document.getElementById("nota").value;
                var resultado;
            }
        </script>
    • Paso 5

      • Ahora, procedemos a evaluar la primer condicion.
        Mostrar "Aplazado", si la nota está entre 1 y 3 (inclusive)

        Siempre que hablemos de condicion, usaremos el comando if.
        Sabemos que if ejecuta su bloque de codigo si la condicion entre () es verdadera.
        ¿Cual sería la condición en este caso?


        Que la nota sea mayor que 0 y menor a 4.
        Para evaluar esto vamos a usar los operadores de comparacion:
        >
        <


        Ver resultado
        nota > 0
        nota < 4
        //Esto suelto no nos sirve, seguir mirando los demas pasos.
    • Paso 6

      • Ya sabemos que las condiciones son nota > 0 Y nota < 4
        Ambas condiciones tienen que ser verdaderas. Vamos a necesitar entonces usar operadores logicos, mas precisamente && (AND).


        Ver resultado
        nota > 0 && nota < 4
        //Esto suelto no nos sirve, seguir mirando los demas pasos.
    • Paso 7

      • Ahora ya sabemos que la condicion del if sera:
        nota > 0 && nota < 4
        Lo que debemos hacer, en base a esto, es asignarle el valor correspondiente a resultado cuando esto sea true.


        Ver resultado
        <script>
            function mostrarResultado(){
                var nota = document.getElementById("nota").value;
                var resultado;
                if(nota > 0 && nota < 4) {
                     resultado = "Aplazado";
                }
            }
        </script>
    • Paso 8

      • Vamos a evaluar la siguiente condicion.
        Mostrar "Desaprobado", si la nota está entre 4 y 6 (inclusive)
        Siguiendo la misma logica, crearemos un nuevo if, asignandole el valor "Desaprobado" si la nota es > 3 y < 7


        Ver resultado
        <script>
            function mostrarResultado(){
                var nota = document.getElementById("nota").value;
                var resultado;
                if(nota > 0 && nota < 4) {
                     resultado = "Aplazado";
                }
                if(nota > 3 && nota < 7) {
                     resultado = "Desaprobado";
                }
            }
        </script>
    • Paso 9

      • Vamos a evaluar la siguiente condicion.
        Mostrar "Aprobado", si la nota está entre 7 y 10 (inclusive)
        Siguiendo la misma logica, crearemos un tercer if, asignandole el valor "Aprobado" si la nota es > 6 y < 11


        Ver resultado
        <script>
            function mostrarResultado(){
                var nota = document.getElementById("nota").value;
                var resultado;
                if(nota > 0 && nota < 4) {
                     resultado = "Aplazado";
                }
                if(nota > 3 && nota < 7) {
                     resultado = "Desaprobado";
                }
                if(nota > 6 && nota < 11) {
                     resultado = "Aprobado";
                }
            }
        </script>
    • Paso 10

      • Por ultimo, vamos a mostrar el contenido de la variable resultado en un alert


        Ver resultado
        <script>
            function mostrarResultado(){
                var nota = document.getElementById("nota").value;
                var resultado;
                if(nota > 0 && nota < 4) {
                     resultado = "Aplazado";
                }
                if(nota > 3 && nota < 7) {
                     resultado = "Desaprobado";
                }
                if(nota > 6 && nota < 11) {
                     resultado = "Aprobado";
                }
                 alert(resultado);
            }
        </script>
Segundo ejercicio - Conversor de moneda
  • Consigna

    Debemos crear un programa que pida un valor en dolares mediante un input, y muestre su cotizacion en pesos en pantalla (no con un alert).
    No usaremos buttons. También deberemos mostrar un error con un alert si lo ingresado no es un número.

      Para esto crearemos:

    • ⚫ Una carpeta llamada conversor
    • ⚫ Dentro de conversor, crearemos un archivo index.html
    • ⚫ Trabajaremos con jQuery, asi que incluimos el archivo jquery.js dentro de la carpeta conversor
  • Paso a paso
    • Paso 1

      • En index.html escribimos la estructura básica de la página web.
        Agregamos dos tag script en el head. Al primero le ponemos el atributo src para vincular jQuery al proyecto.


        Ver resultado
        <html>
            <head>
                 <script src="jquery.js" > </script> // Verificar que tu jquery se llame igual

                <script>

                 </script>
            </head>

            <body>

            </body>
        </html>
    • Paso 2

      • Dentro del body, crearemos un tag h1 para pedirle al usuario que ingrese un valor en dolares, y un tag input donde deberá ingresarlo.
        Este ultimo debe tener un atributo id que sea dolar.

        Tambien se creara un h1 con un id que sea resultadoPesos


        Ver resultado
        <body>
            <h1> Ingrese un valor en USD: </h1>

            <input id="dolar" />
            <h1 id="resultadoPesos"> </h1>
        </body>
    • Paso 3

      • Dentro del script, empezaremos a programar
        Arrancaremos con crear una funcion que se llame convertir


        Ver resultado
        <script>
            function convertir(){

            }
        </script>
    • Paso 4

      • Dentro de convertir vamos a guardar en una variable el contenido del input (el valor en dolares). Tambien vamos a crear una variable llamada cotizacion que tendra de valor 80.
        Para tomar el valor del input vamos a usar jQuery. Recordamos que la sintaxis para obtener el valor de un input es $("#idDelInput").val()


        Ver resultado
        <script>
            function convertir(){
                var dolares = $("#dolar").val();
                var cotizacion = 80;
            }
        </script>
    • Paso 5

      • Vamos a crear una variable pesos que tendra como valor la multiplicación entre dolar y cotizacion.


        Ver resultado
        <script>
            function convertir(){
                var dolares = $("#dolar").val();
                var cotizacion = 80;
                var pesos = dolares * cotizacion;
            }
        </script>
    • Paso 6

      • Ahora vamos a mostrar el resultado en pantalla.
        En lugar de hacerlo con un alert() como lo hacemos habitualmente, esta vez editaremos una etiqueta de HTML mediante jQuery.
        La sintaxis es la siguiente:
        $("#idDeLaEtiqueta").text("Aca va el texto que quiero poner!");

        En este caso, la etiqueta que queremos editar es resultadoPesos y el texto que queremos mostrar es el contenido de la variable pesos


        Ver resultado
        <script>
            function convertir(){
                var dolares = $("#dolar").val();
                var cotizacion = 80;
                var pesos = dolares * cotizacion;
                $("#resultadoPesos").text(pesos);
            }
        </script>
    • Paso 7

      • Ya tenemos nuestra funcion terminada pero aun ningun componente la invoca para que su codigo de ejecute. Habitualmente usamos el evento "onclick" en la etiqueta button.

        Ahora conoceremos un nuevo evento: onkeydown.
        Sabemos que onclick ejecuta la funcion dada, como dice su nombre, cuando se hace click. En cambio, onkeydown ejecuta la funcion dada cuando una tecla es pulsada.

        onkeydown va en los inputs, a diferencia de onclick que va en los buttons. La sintaxis es la misma:
        onkeydown="miFuncion()"
        Ahora vamos a agregar el evento onkeydown en el input con id dolar.


        Ver resultado
        <body>
            <h1> Ingrese un valor en USD: </h1>

            <input onkeydown="convertir()" id="dolar" />
            <h1 id="resultadoPesos"> </h1>
        </body>
